# budgetos

The personal finance operating system.

## Features

- **Dashboard** — Net Worth and Cash overview
- **Accounts** — Manage checking, savings, credit cards, investments, crypto, and cash accounts

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- React Router
- FinanceContext for state management
